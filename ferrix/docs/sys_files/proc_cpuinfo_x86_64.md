# `/proc/cpuinfo` (x86_64)

Этот файл содержит основную информацию о процессоре(ах), установленном(ых) в компьютер. Содержит ряд параметров в формате `параметр: значение\n`; если в системе установлен многоядерный/многопоточный процессор, то параметры по каждому ядру и/или потоку объединяются в группы, в качестве разделителя группы используется пустая строка.

## Основные параметры

| Имя параметра | Краткое описание |
|---------------|------------------|
| `processor` | Порядковый номер логического процессора (ядро/поток) |
| `vendor_id` | Производитель процессора (`GenuineIntel`, `AuthenticAMD`, etc.) |
| `cpu family` | Семейство процессора (например, `6` для Intel Core, `23` для AMD Zen) |
| `model` | Модель ЦП в рамках семейства (также число) |
| `model_name` | Имя модели ЦП |
| `stepping` | Ревизия ядра |
| `microcode` | Шестнадцатеричное значение -- версия микрокода |
| `cpu MHz` | Текущая частота ЦП в МГц |
| `cache_size` | Размер кеша L3 |
| `physical_id` | Номер физического сокета/процессора |
| `siblings` | Количество логических процессоров в одном физическом сокете |
| `core id` | Идентификатор физического ядра внутри сокета |
| `cpu cores` | Количество физических ядер в сокете |
| `apicid` | Идентификатор APIC (Advanced Programmable Interrupt Controller) |
| `initial apicid` | Начальный APIC ID |
| `fpu` | Наличие FPU |
| `fpu_exception` | Поддержка исключений FPU |
| `wp` | Поддержка write-protect в режиме супервизора |
| `flags` | Список поддерживаемых инструкций и технологий |
| `bugs` | Баги и уязвимости (более подробную инфо см. на странице "CPU Vulnerabilities" ("Уязвимости ЦП")) |
| `bogomips` | Приблизительный показатель быстродействия, который на данный момент устарел |
| `clflush size` | Размер строки кеша для инструкции CLFLUSH |
| `cache_alignment` | Выравнивание кеша |
| `address sizes` | Разрядность физических и виртуальных адресов |
| `power management` | Поддержка энергосберегающих технологий |

### Значения параметра `flags`

| Инструкции/Технологии | Возможные значения |
|-----------------------|--------------------|
| SIMD | `mmx`, `sse`, `sse2`, `ssse3`, `sse4_1`, `sse4_2`, `avx`, `avx2` |
| Intel VT-x | `vmx` |
| AMD-V | `svm` |
| Аппаратное ускорение AES | `aes` |
| Time Stamp Counter | `tsc` |
| Аппаратный ГСЧ | `rdrand` |
| Безопасность | `smx`, `sgx` |
| Наличие гипервизора | `hypervizor` |
